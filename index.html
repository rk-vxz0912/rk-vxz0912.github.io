Services

お手伝いできること

スポットでの相談から、継続的な開発・運用まで対応可能です。 まずは小さく始めて、良ければ段階的に広げていく形を想定しています。

Webアプリの新規開発

要件整理〜設計・実装・デプロイまで、一通り対応します。 認証付きの管理画面や、社内向けツールなどを想定しています。

Tech例：FastAPI / Django / Next.js / PostgreSQL / Docker

既存システムの改善・リファクタリング

既存のコードを読み解きつつ、バグ修正や機能追加、パフォーマンス改善などを行います。 ちょっと触る人がいなくて困っている、といったケースも歓迎です。

小さな改修から長期の保守フェーズまで対応可能

AWSインフラ設計・構築

小規模〜中規模サービス向けの VPC / EC2 / RDS / ALB などの構成検討と構築を行います。 セキュリティや運用も含めて一緒に考えます。

IaC（CloudFormation / Terraform）にも対応可能（要相談）

CI/CD・自動化の整備

GitLab CI / GitHub Actions を使ったビルド・テスト・デプロイの自動化パイプラインを構築します。 手作業の運用フローを減らし、ミスの起きにくい体制づくりを支援します。

既存パイプラインの改善・トラブルシュートのみの相談もOKです。
